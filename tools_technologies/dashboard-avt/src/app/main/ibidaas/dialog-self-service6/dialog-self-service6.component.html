Decision Tree

Description:

A decision tree is a supervised machine learning algorithm that can be used for both classification and regression problems. The goal of using a Decision Tree is to create a training model that can use to predict the class or value of the target variable by learning simple decision rules inferred from prior data (training data).

Parameters:
  • Attribute of selection measure (e.g gini, entropy)
  • Maximum depth of a tree
When to use:
  • If you have labeled data
  • If the output of the model is a class (classification)
  • If the output of the model is a number (regression)
