K-means Evaluation

Description:

K-means algorithm is a supervised learning algorithm which is used for both classification as well as regression. In evaluation, clustering results are evaluated based on data that was clustered itself and on data not used for clustering, such as known class labels and external benchmarks. These types of evaluation methods measure how close the clustering is to the predetermined benchmark classes.

Parameters:
  • Number of clusters: k
When to use:
  • If you have labeled data
  • If the output of the model is a class (classification)
  • If the output of the model is a number (regression)
