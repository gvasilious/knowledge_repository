K-means Prediction

Description:

K-means algorithm is a supervised learning algorithm which is used for both classification as well as regression. To process the learning data, the K-means algorithm in data mining starts with a first group of randomly selected centroids, which are used as the beginning points for every cluster, and then performs iterative (repetitive) calculations to optimize the positions of the centroids.

Parameters:
  • Number of clusters: k
When to use:
  • If you have labeled data
  • If the output of the model is a class (classification)
  • If the output of the model is a number (regression)
