# Differential Privacy K-Means clustering implementation based on COMPSs framework

**Running - run as:**

enqueue_compss  --num_nodes=n --python_interpreter=python3 dpkmeans_example.py

where n is the number of nodes used.

**Required software:**

- COMPSs framework installed on a HPC cluster
- Python3

**Required python packages:**

- PyCOMPSs
- numpy
- scipy
- sklearn

**Implementations:**
- dpkmeans.py - The Differential Privacy K-Means Clustering Class


**Result: **
- Centroids of clusters
- Number of iterations performed
